# International One-Day Cricket Dataset (2001–2025)

A structured ball-by-ball dataset covering **3,073 One-Day International (ODI)** cricket matches played between **2001 and 2025**.
This dataset provides detailed delivery-level events, player actions, and match metadata—ideal for analytics, research, visualization, and machine learning projects.

## Features
- 3,073 ODI matches
- Ball-by-ball delivery events
- Runs, wickets, extras, dismissals
- Batting, bowling, and fielding details
- Match metadata (teams, venue, date, toss, result)

## Format
Available in **JSON** and **YAML** versions.
